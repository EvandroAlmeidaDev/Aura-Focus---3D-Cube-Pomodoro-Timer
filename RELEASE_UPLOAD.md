# Publicar release no GitHub

Para disponibilizar o executável para download:

1. Acesse: **https://github.com/EvandroAlmeidaDev/Aura-Focus---3D-Cube-Pomodoro-Timer/releases**
2. Clique em **"Draft a new release"** (ou "Create a new release").
3. Em **"Choose a tag"**, digite por exemplo `v4.0.0` e clique em **"Create new tag: v4.0.0 on publish"**.
4. **Release title:** `Aura Focus v4.0.0` (ou o título que preferir).
5. **Description:** pode colar algo como:
   ```
   ## Aura Focus 4.0.0
   Timer Pomodoro com cubo 3D.
   - Windows: baixe o arquivo **win-unpacked.zip**, extraia e execute `Aura Focus.exe`.
   ```
6. Em **"Attach binaries"**, arraste o arquivo:
   `G:\aura-focus-tauri\release\win-unpacked.zip`
   ou clique e selecione esse arquivo.
7. Clique em **"Publish release"**.

O link de download ficará em:  
https://github.com/EvandroAlmeidaDev/Aura-Focus---3D-Cube-Pomodoro-Timer/releases
